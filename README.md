# e-plantShopping

e-plantShopping is a React-based online plant shopping application developed as part of the Paradise Nursery project.

The application allows users to:

* Browse houseplants by category.
* View plant details including name, image, and price.
* Add plants to a shopping cart.
* Manage cart items by increasing, decreasing, or removing quantities.
* View the total number of items in the cart.

This project uses React and Redux Toolkit to provide a responsive and interactive shopping experience.
